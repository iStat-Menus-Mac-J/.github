## Overview

iStat Menus Advanced is a comprehensive system monitoring application from Bjango that displays real-time CPU, GPU, memory, disk, network, battery, and sensor data directly in the menu bar, providing continuous visibility into system performance without opening a separate monitoring application. It transforms the menu bar into an at-a-glance system health dashboard that keeps critical performance metrics visible during every task. The application has been the standard system monitor for power users and professionals for over fifteen years, refined through continuous development to balance information density with visual clarity.

CPU and GPU monitoring shows per-core utilization, process-level resource consumption, frequency, temperature, and historical usage graphs that reveal performance patterns over time and identify processes that consume excessive resources. The dropdown menu displays a ranked list of the most resource-intensive processes with their CPU percentage, making it immediately clear what is consuming processing power. Historical graphs show trends over the past hour, day, or week, revealing patterns that momentary monitoring would miss.

Memory and disk monitoring tracks RAM usage breakdown by category — active, wired, compressed, inactive — alongside disk activity rates, available storage, and SMART health status for connected drives. Memory pressure indicators distinguish between high memory usage that is normal caching behavior and genuine memory pressure that degrades performance. Disk temperature monitoring for drives that report thermal data provides early warning of hardware stress.

Network monitoring displays real-time upload and download speeds, connection type, IP addresses, and per-process network usage that identifies which applications are consuming bandwidth. The dropdown shows active connections with remote addresses and ports, providing network visibility that helps diagnose connectivity issues and identify unexpected network activity. Historical bandwidth graphs track usage over configurable periods.

For power users, developers, and system administrators who want continuous visibility into system performance without dedicating screen space to a monitoring application, iStat Menus Advanced is the system monitor that makes comprehensive hardware metrics available at a glance from the menu bar. Its information density, visual polish, and fifteen years of refinement make it the definitive system monitoring tool for anyone who cares about understanding how their hardware is performing.

## Performance

iStat Menus balances performance with system resource usage intelligently. Operations are handled in the background, keeping your system responsive for everything else. Efficient memory management and smart caching reduce overhead during extended work sessions.

Apple Silicon acceleration is fully utilized on M-series chips, delivering noticeably faster performance. On Intel machines, hardware acceleration is used wherever available.
